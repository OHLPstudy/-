# 스터디 방법
- 각 주차 스터디원이 제출한 퀴즈 풀이
- 2시간 풀이 후 1시간 설명
- 문제는 이론문제 (서술x) 1개, 코딩문제 1개
- **문제 자체를 ChatGPT에 물어보기 금지 **

- 11월 9일 : 2 ~ 5
