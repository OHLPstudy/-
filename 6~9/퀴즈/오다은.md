## 코드문제
1. 오버로딩과 오버라이딩 개념을 활용하여 다음 코드를 실행했을 때 출력 결과를 쓰시오.
   또한, 코드 실행 중 어떤 메서드가 호출되는지 이유를 포함해 분석하시오.
   
   ```java
   class Parent {
        void print(int a) {
            System.out.println("Parent class: " + a);
        }
    
        void display() {
            System.out.println("Display method in Parent");
        }
    }
    
    class Child extends Parent {
        void print(String str) {
            System.out.println("Child class (Overloaded): " + str);
        }
    
        @Override
        void print(int a) {
            System.out.println("Child class (Overridden): " + a);
        }
    
        void display(String msg) {
            System.out.println("Display method in Child: " + msg);
        }
    }
    
    public class Main {
        public static void main(String[] args) {
            Parent obj1 = new Parent();
            Parent obj2 = new Child();
            Child obj3 = new Child();
    
            obj1.print(10);
            obj2.print(20);
            obj3.print("Hello");
            obj3.print(30);
    
            obj1.display();
            obj2.display();
            obj3.display("Overloaded");
        }
    }
    ```
